# Gaming Image Sources (No Hotlink Blocking)

## Recommended Sources:

### 1. RAWG.io (Best for Gaming)
- **URL Pattern**: `https://media.rawg.io/media/games/[id]/[hash].jpg`
- **Pros**: Free, no hotlink blocking, huge gaming database
- **Example**: Elden Ring - `https://media.rawg.io/media/games/5ec/5ecac5cb026ec26a56efcc546364e348.jpg`

### 2. Local Images (Best for School Projects)
- Store images in `ff/images/` folder
- Reference as: `/images/game-name.jpg`
- **Pros**: Full control, no external dependencies
- **Cons**: Need to download images first

### 3. ImgBB.com
- Upload images at imgbb.com
- Get direct link
- **Pros**: Free, simple, reliable
- **Cons**: Manual upload required

### 4. Steam CDN (Official Game Images)
- **URL Pattern**: `https://cdn.cloudflare.steamstatic.com/steam/apps/[appid]/header.jpg`
- **Example**: Elden Ring (1245620) - `https://cdn.cloudflare.steamstatic.com/steam/apps/1245620/header.jpg`
- **Pros**: Official, high quality
- **Cons**: Need to find Steam App ID

## Current Setup:
Your products now use **RAWG.io** images which are:
- ✅ Free to use
- ✅ No hotlink blocking
- ✅ High quality gaming images
- ✅ Reliable and fast

## To Add More Images:
1. Visit https://rawg.io
2. Search for a game
3. Right-click on game image → Copy Image Address
4. Paste URL in admin panel when editing product
