# How to Add Local Images

## Quick Start:

1. **Download game images** from any source (Google Images, game websites, etc.)

2. **Save them** in this folder: `ff/images/`
   - Example: `ff/images/elden-ring.jpg`

3. **Update product in Admin Panel**:
   - Go to admin panel
   - Click edit (blue pencil icon) on a product
   - In "Image URL" field, enter: `/images/elden-ring.jpg`
   - Click "Update Product"

## File Naming Tips:
- Use lowercase
- Use hyphens instead of spaces
- Example: `the-last-of-us.jpg` not `The Last of Us.jpg`

## Supported Formats:
- .jpg / .jpeg
- .png
- .gif
- .webp

## Example:
If you have a file: `ff/images/fifa-24.jpg`
Then in admin panel, use: `/images/fifa-24.jpg`

That's it! The image will display on your site.
